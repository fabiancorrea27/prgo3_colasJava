# Demostracion Colas

Programa que permite observar el funcionamiento de colas.

## Requisitos

- Java JDK 17 (seguro)

## Instrucciones de uso

- Primero ingresar dinero
- Dar siguiente para mostrar a la siguiente persona en la cola y restar el dinero al cajero
